  • X-Clone

    React.js, Node.js, Express.js, MongoDB, TanStack Query and Tailwind CSS.

    Developed a full-stack social media application, X Clone, using React.js, Node.js, Express.js, MongoDB, and Tailwind CSS, implementing JWT authentication and bcrypt.js for secure user sessions. Built core social features including posts, likes, comments, real-time notifications, user suggestions, and dynamic profile editing with image uploads via Cloudinary. Followed MVC architecture with reusable components, RESTful APIs, and integrated React Query for optimized data fetching and app responsiveness.

  • Note Taking App

    React.js, Node.js, Express.js, MongoDB, TanStack Query and Tailwind CSS.

    Built a full-stack MERN notes app allowing users to create, update, and delete notes with a fully functional REST API. Features include rate limiting with Upstash Redis, a completely responsive UI, exploration of HTTP methods and status codes, and a deployment guide for real-world application experience.

  • Expense Tracker

    MongoDB, Express.js, React.js, and Node.js

    Developed a full-stack MERN expense tracker with real-time analytics, secure JWT authentication, and a responsive Tailwind CSS UI. Features include downloadable Excel reports, interactive charts with date filtering, and toast notifications for an enhanced user experience.

  • Movie App

    ReactJS , Tailwind , Appwrite

    MovieVerse is a smart movie discovery app built with Appwrite and React that lets users search and explore the latest films. It features an intelligent search system that tracks user searches and automatically updates a “Trending Now” section with the top 5 most-searched movies. The app integrates with the TMDB API to display real-time movie details, ratings, and posters in a sleek, responsive interface.

  • Product Store

    React.js, Node.js, Express.js, MongoDB, Chakra UI

    Built a full-featured product store with a responsive React.js and Chakra UI frontend, backed by a Node.js and Express.js API with MongoDB for data management. The store includes robust error handling, smooth deployment, and many interactive features to enhance the user experience.
